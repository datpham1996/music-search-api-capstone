# Music Search Api Capstone (to do now)
This capstone project will be designed to help a user search for event venues based on location and event location types while displaying 
useful details about the venue. 

## 1. Working Prototype (to do now)
You can access a working prototype here: https://nikosamofa.github.io/venue-search-api-capstone


## 2. User Stories 
This app is for a single type of user: a visitor

#### Landing Page (Importance - High)(est:2h)
* as a visitor
* I want to understand what I can do with this app
* so I can decide if I want to use it

#### Main Page (Importance - High)(est:4h)
* as a visitor
* I want to search for track,artist, and album
* so I can decide what I want to listen

#### Result Section (Importance - High)(est:1h)
* as a visitor
* I want to preview a list search result(tracks,artist album cover)
* so I can make a decision based on the information returned

## 3. Functionality 
The app's functionality includes:
* each user should land on the homepage of the application to understand what is the application is about
* each user should be able to search for track, artist and album
* each user should be able to click on search result to see whats in there

## 4. Technology 
* Front-End: HTML5, CSS3, JavaScript, jQuery, Api
* Development Environment: Github
